# Frontend Development

Use this skill for user-facing UI work.

## Workflow

1. Inspect the existing frontend structure, styling system, and component conventions.
2. Make the requested experience usable on the first screen rather than building a marketing placeholder.
3. Keep layout stable with responsive constraints.
4. Preserve existing design patterns unless the user asks for a redesign.
5. Run the relevant build or typecheck command when available.
6. Summarize the visible behavior changed.

Avoid decorative filler, unclear controls, and text that overflows its container.
